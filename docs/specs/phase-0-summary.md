# Phase 0 总结：需求锚点（方向修订）

## 做了什么

- 多轮对齐后形成当前产品方向（**以本文件 + 最新 requirements/design 为准**）：
  1. **原生首页**，无 embed  
  2. **星空主题**；浅色蓝白、深色黑蓝紫  
  3. **用量排行 / 模型健康** → **跳转**既有页（`/rankings`、`/pricing`），首页不嵌大表/监控矩阵  
  4. **跳转双前端路径映射**  
  5. **友链**：系统设置配置，**悬浮球**展示  
  6. 登录可见用户消耗数据的策略仍适用于排行页/API，而非首页大表  
  7. 禁止 UserID 查询
